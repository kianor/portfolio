# Portfolio Kiano Rits

Persoonlijke portfolio-website gemaakt voor het olod **Webtechnologie intro S2** (AP Hogeschool Antwerpen, graduaat SNB).

## Inhoud

- `index.html` — homepagina met hero-carousel, "Over mij" en recente projecten
- `portfolio.html` — overzicht van zes projecten
- `Projecten/dutchcraft.html`, `Projecten/youtube.html`, `Projecten/portfolio.html` — detailpagina's
- `contact.html` — contactformulier
- `css/` — `reset.css`, `fonts.css`, `style.css`
- `assets/images/` — afbeeldingen

## Technologie

- HTML5 met semantische lay-out-elementen (`header`, `nav`, `main`, `section`, `aside`, `footer`)
- Bootstrap 5 voor het responsief raster en componenten
- Eigen CSS in dark-mode stijl bovenop Bootstrap
